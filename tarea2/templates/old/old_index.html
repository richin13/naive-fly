Estilos Clásico

En este formulario se busca determinar el estilo de aprendizaje del visitante, se solicita la misma información que en el sitio de multimedios, sin embargo, no se utiliza la fórmula de las cólumnas sino que se comparan los datos ingresados con los datos históricos que posee la aplicación.

Recinto

Se pretende determinar el lugar o recinto de origen del visitante. Para esto se solicitan tres parámetros: Estilo de aprendizaje, Sexo y Promedio de Matrícula. De igual forma, estos datos son comparados con los datos existentes en la base de datos.

Sexo

En este caso se pretende determinar el sexo del visitante. Para esto se solicitan tres parámetros: Estilo de aprendizaje, Recinto de origen y Promedio de Matrícula. De igual forma, estos datos son comparados con los datos existentes en la base de datos.

Estilos

Nuevamente se pretende determinar el estilo de aprendizaje del visitante. En esta ocasión sólamente se solicitan tres parámetros: Recinto de origen, Sexo y Promedio de matrícula.

Profesor

Con este formulario se pretende determinar el nivel o tipo de un profesor de acuerdo a sus datos de experiencia y conocimiento.

Redes

Con este formulario se pretende determinar el tipo de una red de acuerdo a su capacidad y tamaño.
